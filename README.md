# NestJS Basic CRUD

## Description
A simple CRUD application built with NestJS to manage a resource.

## What's Built
- Basic CRUD operations (Create, Read, Update, Delete)

## How to Run
1. Clone the repository: `git clone https://github.com/yourusername/nestjs-basic-crud.git`
2. Navigate into the directory: `cd nestjs-basic-crud`
3. Install dependencies: `npm install`
4. Start the application: `npm run start`
5. Access the API at `http://localhost:3000/items`

## API Documentation
- **GET /items**: Retrieve all items
- **POST /items**: Create a new item
- **GET /items/:id**: Retrieve a specific item by ID
- **PUT /items/:id**: Update a specific item by ID
- **DELETE /items/:id**: Delete a specific item by ID

## Environment Variables
No specific environment variables are required.
